#### Phase 1 Stack
- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express + MongoDB
- AR: AR.js + Three.js
- Deployment: Vercel + MongoDB Atlas

- Add: Redis caching

- Add: Basic ML pipeline (Python)
